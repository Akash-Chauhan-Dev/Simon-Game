# Simon-Game
A browser-based Simon game built with HTML, CSS, and JavaScript. Follow the pattern, level up, and test your memory!

🎮 Features

🎨 Clean & Interactive UI — Smooth, responsive design for a seamless experience

✨ Animations — Satisfying visual feedback on every button press

🔊 Sound & Music System — Unique sounds for each color make the game more immersive and enjoyable

🏆 High Score Tracking — Compete with yourself and beat your personal best

🛠️ Technologies Used

    HTML  
    CSS  
    JavaScript  


🚀 How to Run

  Clone the repository
    
    git clone https://github.com/Akash-Chauhan-Dev/simon-game.git
    
  Navigate to the project folder
    
    cd simon-game

Open the game
Simply open index.html in your browser — no installation or dependencies needed!    
Or, if you have VS Code, use the Live Server extension for the best experience:

Right-click index.html → Open with Live Server


Start Playing! 🎮
Press any key or click the screen to begin. Follow the flashing sequence and repeat it — good luck! 🏆
